# Benchmark: List Comprehension

## Input Code

```python
squares = [x**2 for x in range(10) if x % 2 == 0]
```

## Expected Behaviors

The explanation MUST:
1. Identify this as Python
2. Mention list comprehension
3. Explain it creates squares of numbers
4. Note it filters for even numbers only
5. Specify the range is 0-9 (or equivalent)

## Scoring

- 1.0: All 5 behaviors present
- 0.5: 3-4 behaviors present
- 0.0: 2 or fewer behaviors present
